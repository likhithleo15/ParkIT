ParkIT: SMART PARKING LOT MANAGER

Developed a full-stack automated parking management system using Flask and YOLOv8 for real-time vehicle detection and gate control.

Integrated Tesseract OCR for Automatic Number Plate Recognition (ANPR) to automate arrival and departure logs.

Designed a relational database using SQLite to manage 200+ parking slots, visitor registrations, and historical activity tracking.

Implemented manual override and CSV export features for administrative reporting and system auditing.

Technologies: Python, Flask, YOLOv8 (Computer Vision), OpenCV, Tesseract OCR, SQLite, HTML/CSS.
